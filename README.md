<p>AI tracking for ultimate firsbee. Maps players onto 2D map and allows for advance analysis through the introduction of an Inertial Measurement Unit (IMU)</p>
<p>For the detection model, model was trained on a pretrained YOLO11 model. The best model up to now would be ultiV1.pt inside the models folder.</p>
<p>Tracking demo using ByteTrack can be found <a href="https://drive.google.com/drive/folders/1PZF5PtyN_IFDd7E4TubW9hClC-6RR6KJ?usp=sharing">here</a>.</p>
